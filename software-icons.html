SOFTWARE ICONS

UI Design

To improve usability, INVIA needed their 4DM software icons to be responsive. They liked the look of the current icons but they weren’t vector format and over time the symbols meaning became inconsistent. In collaboration with another designer we sifted through all 100 icons discussing each icons purpose and establishing a set of rules. We didn't want to change too many icons because the users were familiar with the current symbols, so we only did a complete change if we thought it would improve readability.
